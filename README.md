# app-myOng
Aplicación para gestionar de una asociación. La asociacion tiene socios. 
De cada socio vamos a guardar datos personales.

## Modelo socio
* Identificador
* DNI
* [X]Nombre
* [X]Apellidos
* [X]Fecha de nacimiento 
* [X]Teléfono
* [X]E-mail
* Dirección:

* [X]IBAN (Opcional)
* [X]Fecha de alta.
* Tutor si es menor de edad
